Design System Guide

This design system provides all components needed to build the KodNest Premium Build System. Every element follows the calm, intentional design philosophy.

Key Principles

  • Use only spacing values from the scale
  • Maximum 4 colors across entire system
  • Serif for headings, sans-serif for body
  • 150-200ms transitions, ease-in-out
  • No gradients, no glassmorphism, no neon
